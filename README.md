# HiveQL Repository

This repository contains resources and examples for working with HiveQL and Spark DataFrames for data analysis and processing.

## Overview

This repository provides practical examples of data manipulation, transformation, and analysis using Spark DataFrames and SQL. It demonstrates how to load, process, and analyze e-commerce datasets including orders, payments, customers, products, and more.

## Contents

The repository is organized with the following structure:

## Features

- **Data Loading**: Examples of loading CSV data into Spark DataFrames
- **Schema Exploration**: Methods to examine and understand data structure
- **Data Analysis**: Techniques for counting, aggregating, and analyzing e-commerce data
- **SQL Integration**: Examples of using SQL queries with Spark DataFrames

## Dataset

The examples work with an e-commerce dataset that includes:

- Orders data
- Payment information
- Customer records
- Product details
- Reviews
- Order items
- Seller information
- Geolocation data
- Category translations

## Getting Started

To use this repository:

1. Clone the repository to your local machine
2. Open the Jupyter notebooks in a Spark-enabled environment
3. Follow the examples to learn how to manipulate data using Spark DataFrames and HiveQL

## Example Usage

The notebooks demonstrate various data operations, including:

- Loading data from CSV files
- Displaying schema information
- Performing count operations on unique orders and customers
- Using SQL functions with DataFrames

## Requirements

- Apache Spark
- Python with PySpark
- Jupyter Notebook environment
